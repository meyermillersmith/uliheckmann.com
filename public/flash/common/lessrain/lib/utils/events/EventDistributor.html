distributeEvent

Usage

public function distributeEvent(eventObject:IEvent) :Void

Description

distributes an EventObject to specific Listeners of EventDistributor. Which Listeners receive the event is decided via getType() method of the DistributorEvent. Furthermore the DistributorEvent is distributed along with the IEvent implementing eventObject to the listeners, too to provide an easy way to remove the listener.

Parameters

eventObject
must implement IEvent

Returns

Nothing.
